Desactivar la expiración de contraseñas

12 Sep 2013

Con la utilidad chage podemos reasignar los valores de expiración.

Verificamos las fechas de expiracion de los passwords

$ chage -l usuario

Last password change                                    : Jul 23, 2012
Password expires     : Sep 21, 2012
Password inactive     : Sep 28, 2012
Account expires      : never
Minimum number of days between password change  : 1
Maximum number of days between password change  : 60
Number of days of warning before password expires : 7

Tenemos dos modos para poder modificar la expiración:

  • Modo interactivo. En prompt iras escribiendo los valores que se te solicitan
$ chage usuario
  • Modo comando. Restablecer valores por defecto.
$ chage -I -1 -m 0 -M 99999 -E -1 usuario
