Plugin Java en Chrome y Firefox en tu Linux

18 Apr 2014

Normalmente se me olvida configurar este plugin Java, espero les sea útil como a mi tener una publicación para recordarlo. Supongamos que ya tienen instalado alguna versión del JRE ó JDK de Java, continué con la siguiente configuración:

Google Chrome

Creamos un directorio denominado plugins si aún no existe.

$ sudo mkdir -p /opt/google/chrome/plugins

Nos cambiamos al directorio de plugins creado.

$ cd /opt/google/chrome/plugins

Creamos un enlace simbólico.

$ sudo ln -s /usr/java/jre1.7.0_55/lib/amd64/libnpjp2.so

Tome en cuenta que el directorio donde se encuentra libnpjp2.so puede variar según su configuración Java. Para localizarlo podemos ejecutar locate y encontrar el ó los directorios donde este el plugin.

$ sudo updatedb
$ locate libnpjp2.so
/usr/java/jdk1.6.0_43/jre/lib/amd64/libnpjp2.so
/usr/java/jre1.7.0_21/lib/amd64/libnpjp2.so
/usr/java/jre1.7.0_55/lib/amd64/libnpjp2.so

Tengo tres directorios actualmente, configurare con el mas reciente /usr/java/jre1.7.0_55/lib/amd64/libnpjp2.so.

Reiniciamos el explorador

Mozilla Firefox

Creemos un directorio denominado plugins si aún no existe.

$ sudo mkdir -p /usr/lib/mozilla/plugins

Ó si estas usando S.0 de 64bits

$ sudo mkdir -p /usr/lib64/mozilla/plugins

Nos cambiamos al directorio de plugins creado.

cd /usr/lib/mozilla/plugins

Ó si estas usando S.0 de 64bits

$ cd /usr/lib64/mozilla/plugins

Creamos un enlace simbólico.

$ sudo ln -s /usr/java/jre1.7.0_55/lib/amd64/libnpjp2.so

Reiniciamos el explorador
