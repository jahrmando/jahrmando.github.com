Obtener una lista de usuarios conectados en SMB Samba

07 Apr 2014

Sabemos que el comando smbstatus nos muestra información acerca del estado actual del servicio samba, como usuarios conectados y ficheros abiertos; con la opción -p podemos tener un filtro de solo procesos y un ejemplo de salida seria:

$ smbstatus -p

Samba version 3.5.10-125.el6
PID     Username      Group         Machine                        
---------------------------------------------------------------
13777     user_01   user_01   machine_01 (192.168.0.161)
9272      user_02  user_02  machine_02 (192.168.0.182)
11400     user_03    user_03    machine_03    (192.168.0.151)

Vemos que la salida contiene la lista de usuarios conectados pero nos falta aplicarle formato para solo extraer la información requerida el usuario y la IP. Para ello usaremos los comandos sed y awk.

En el segundo bloque uso sed para eliminar las primeras 4 lineas de la salida, en el tercer bloque elimino los símbolos ( y ) que contiene las IPs, por ultimo con awk imprimo las columna 2 y 5 con el siguiente formato: la primera columna tipo cadena de 20 caracteres %-20s, una tabulacion \t, una columna tipo cadena %s y una nueva linea \n.

$ smbstatus -p | sed '1,4d' | sed 's/(\|)//g' | awk '{printf "%-20s\t%s\n", $2, $5}'
user_01          192.168.0.161
user_02          192.168.0.182
user_03          192.168.0.151

La salida ahora es una lista de usuario - IP, que de igual forma podemos contar el numero de lineas impresas para determinar la cantidad de usuarios conectados usando wc -l.

$ smbstatus -p | sed '1,4d'| sed 's/(\|)//g' | awk '{printf "%-20s\t%s\n", $2, $5}' | wc -l
3

Ó de forma mas breve, solo conservando el bloque donde eliminamos las primeras 4 lineas:

smbstatus -p | sed '1,4d' | wc -l
