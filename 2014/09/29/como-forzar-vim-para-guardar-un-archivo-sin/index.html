Guardar un archivo sin permisos de escritura en VIM

29 Sep 2014

Se que no somos perfectos y yo en lo particular cometo el error de abrir un archivo para editarlo a la cual no tengo permisos de escritura, y claro! me doy cuenta cuando ya llevo varios minutos trabajando sobre ese archivo, siempre sucede y es inevitable.

Con el fin de que no suceda de nuevo he encontrado este pequeño VIM hack. Creamos un command-line (cmap) VIM de la siguiente forma en el archivo de configuracion local de VIM .vimrc, esta se encuentra en el HOME del usuario.

$ vi .vimrc
" Forzar guardar si requiere privilegios de superusuario 
cmap w!! w !sudo tee > /dev/null %

Este comando toma el input actual y crea un documento con en el mismo nombre y en el mismo directorio del archivo abierto (lo sobrescribe) y usando sudo obtiene privilegios de súperusuario para crearlo.

Puedes encontrar mas información aquí sobre este hack.

Después de esto cuando tengamos problemas para poder escribir nuestros cambios en el documento ejecutamos en modo comando w!!.
