Generar listado de contribuidores en un repositorio GIT

29 Jul 2014

Una forma sencilla de generar una lista completa de correos electronicos de los que han contribuido en un proyecto GIT

$ git log --pretty=format:"%aE" | sort -u    
userA@hotmail.com
userB@gmail.com
userC@outlook.com
userD@gmail.com
