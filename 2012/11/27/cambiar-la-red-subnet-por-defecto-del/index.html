Editar la subnet default de VirtualBox

27 Nov 2012

Cambiar la red (subnet) por defecto del modo NAT de VirtualBox. Verificamos la lista de host invitados registrados

[user@host ~]$ VBoxManage list vms
"CentOS.Server.1" {4d4d80ed-a07e-4a98-98ee-19952ea0ba6c}

Registramos nuestro nuevo host invitado, escribimos el path donde se encuentra nuestro archivo .vbox del host que queremos registrar.

[user@host ~]$ VBoxManage registervm /home/jahrmando/VirtualBox/CentOSServer2/CentOSServer2.vbox
[user@host ~]$ VBoxManage list vms
"CentOS.Server.1" {4d4d80ed-a07e-4a98-98ee-19952ea0ba6c}
"CentOS Server 2" {4b3dc98b-a63f-4794-a508-b63619ff29e4}

Aplicamos el cambio de la subnet que queremos aplicar, en este caso la opcion –natnetX llamese X a la ID de la interfaz de red que esta en modo NAT.

La sintaxis quedaria como: VBoxManage modifyvm [UID] --natnet[n] "subnet"

$ VBoxManage modifyvm 4b3dc98b-a63f-4794-a508-b63619ff29e4 --natnet1 "172.16/16"
