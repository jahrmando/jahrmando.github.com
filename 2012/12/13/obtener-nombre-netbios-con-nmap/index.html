Obtener los nombres de PC Windows de la red

13 Dec 2012

Con NMAP se puede obtener mucha informacion importante de la red local a la que estemos conectados, una de ellas es el nombre de algun equipo Windows en la red.

Se puede conseguir usando los siguientes comandos:

nmap -sU -sS –script smb-os-discovery.nse -p U:137,T:139 IPHOST nmap –script smb-os-discovery.nse -p445 IPHOST

Observaciones

Revisar si tenemos el script smb-os-discovery.nse en el directorio de plugins de nmap, usualmente en el path /usr/share/nmap/scripts/

Ejecute con privilegios root o usando sudo

Ejemplo 1:

$ sudo nmap -sU -sS --script smb-os-discovery.nse -p U:137,T:139 192.168.1.130
